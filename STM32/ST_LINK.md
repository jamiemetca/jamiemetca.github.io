# ST-LINK (embedded ST-LINK/V2-1)
This documentation is related to the [STM32F042K6T6](/STM32/STM32F042K6T6) and
so is concerned with the embedded ST-LINK/V2-1.

The ST-LINK is an embedded in-circuit programming and debugging tool (It's a
chip).
It has a [virtual COM port](/STM32/virtual_com_port) interface on USB
